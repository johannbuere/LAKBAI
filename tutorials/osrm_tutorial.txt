NOTE: -> MAKE SURE TO PUT THE DIRECTORY AT THE C DRIVE FOR THIS TUTORIAL TO WORK. Example: C:\LAKBAI\osrm-docker
      -> This is for car profiles only!


Prerequisites before tutorial:
-> Download albay.osm.pbf from LAKBAI repo(repo natin)
-> Have Docker downloaded and setup

OSRM Installation Process for Windows
(Optional) Visit https://github.com/Project-OSRM/osrm-backend for premade code and documentation

Step 1: Go or make a directory named LAKBAI at C drive
	Example C:\LAKBAI\osrm-docker

Step 2: Move albay.osm.pbf to C:\LAKBAI\osrm-docker directory. MAKE SURE TO BE IN THE C:\LAKBAI DIRECTORY WHEN DOING THE FOLLOWING CODES. 
	NOTE: just do: cd C:\LAKBAI\osrm-docker
 
Step 3: Paste this code to Windows Powershell: docker run -t -v C:/LAKBAI/osrm-docker:/data ghcr.io/project-osrm/osrm-backend osrm-extract -p /opt/car.lua /data/albay.osm.pbf

Step 4: Paste this code next to Windows Powershell: docker run -t -v C:/LAKBAI/osrm-docker:/data ghcr.io/project-osrm/osrm-backend osrm-partition /data/albay.osrm

Step 5: Next, past another code again: docker run -t -v C:/LAKBAI/osrm-docker:/data ghcr.io/project-osrm/osrm-backend osrm-customize /data/albay.osrm

Step 6: Check server if it's up and running on localhost: docker run -t -i -p 5000:5000 -v "C:/LAKBAI/osrm-docker:/data" ghcr.io/project-osrm/osrm-backend osrm-routed --algorithm mld /data/albay.osrm

	-> Note: Make sure code line ends with: "[info] running and waiting for requests"
	-> DO NOT CLOSE THIS TERMINAL. CHECK UNTIL THE END
	

Step 7 (Final): Open a new terminal and past -> curl "https://router.project-osrm.org/route/v1/driving/13.388860,52.517037;13.385983,52.496891?steps=true&alternatives=true"

	-> Note -> Make sure code status shows OK

CLOSING TERMINALS: Go back to the terminal you used at Step 6 and press Ctrl+C. This ends the service. You can now exit docker but make sure to check if there are active containers and images. Close it (NOT DELETE) or pause it.
